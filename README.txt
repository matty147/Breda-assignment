Night Shade
===========
BUas intake assignment 2026-2027 - theme: "Transform"
Made in C++ using the tmpl8 framework.

How to play
-----------
Reach the portal at the end of each level while avoiding enemies and 
other dangerous obstacles. You can change the time of the day to change 
the properties of the world around you. The game contains 3 levels.

Controls
--------
A / Left arrow   - move left
D / Right arrow  - move right
W / Up arrow     - jump
P                - take a screenshot
Esc              - quit

Build
-----
Open NightShade.sln in Visual Studio Community, select Debug or Release (x64), 
and build & run. All dependencies are included in the project.
