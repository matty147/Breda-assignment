# Night shade shifter
BUas intake assignment made in c++ with tmpl8

- download the game from [itch.io](https://inkk-ing.itch.io/night-shade-shifter)
## features
- 3 levels

## Controls
- A/lLeft arrow to go left
- D/Right arrow to go right
- W/Up arrow to jump
- P to take a screenshot
- esc key to exit
